# NLP Framework for Propaganda Detection

## Description

This repository contains the implementation and datasets for an NLP-based framework designed to detect propaganda in text. This project is part of research work focused on developing machine learning models to identify and classify propaganda techniques in social media and news content.

## Datasets Used

This project utilizes the following publicly available datasets:

1. **LIAR-Twitter Dataset**
   - URL: https://www.kaggle.com/datasets/muhammadimran112233/liar-twitter-dataset
   - Contains labeled propaganda examples from Twitter
   - Used for training and evaluating the detection models

2. **LIAR-Twitter Dataset** (Additional reference)
   - URL: https://www.kaggle.com/datasets/muhammadimran112233/liar-twitter-dataset
   - Supplementary dataset for comprehensive model validation

## Contents

All code and analysis are implemented in Jupyter Notebooks, providing interactive documentation of the data processing, model training, and evaluation pipeline.

## Usage

To use this repository:
1. Download the datasets from the Kaggle links provided above
2. Place the dataset files in the appropriate directory
3. Run the Jupyter Notebooks to reproduce the analysis and model training

## Requirements

- Python 3.x
- Jupyter Notebook
- Standard NLP libraries (specified in notebooks)
